# PRD Intake Questions（最大8問）

すでに分かっているものだけ回答でOK。分からないものは「不明」でOK（Assumptionsに回す）。

1. これは誰の、どんな状況の問題？（target customer / context）
2. 何が困っていて、なぜ今やる？（problem statement / why now）
3. 何ができたら成功？（KPI + guardrails / baseline / 計測方法）
4. いつまでに、どのくらいの投入量？（target release / appetite）
5. MVPで必ず入れるMustは3つまでにするとしたら？（scope）
6. 今回“やらないこと”は？（non-goals / out of scope）
7. 一番怖いリスク・依存は？（risks / dependencies）
8. PR/FAQ（顧客から逆算）も必要？（合意形成が難しい/方向性がブレそうなら推奨）
