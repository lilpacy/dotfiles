# Vocabulary Coach (語彙コーチ) — Claude Code Skill

「人間の専門家みたいに、次に調べると良い“語彙”をくれるAI」をClaude Codeで再現するためのスキルです。

## できること
- 問いの枠（確率分布）を意図的に広げる **NE（探索）→TI（評価）** の二段階ワークフロー
- 5観点（技術 / UX / プロダクト / 運用 / ビジネス）での **用語バンク生成**
- 隣接領域への **リフレーミング（別フレーム）**
- 「欲しい効果→用語」への **逆引き（reverse index）**
- 最後に **検索クエリ（日本語 + 英語）** へ落とす

## インストール
1. このzipを解凍
2. リポジトリ直下に `.claude/` を置く（すでにある場合は中身をマージ）

## 使い方
- `/vocab` : 課題から用語を増やす（まずはこれ）
- `/reframe` : 別領域のフレームで言い換えて用語を発掘
- `/reverse-index` : 欲しい効果から用語を逆引き

## 入力のコツ
- 1〜2段落で良いので「状況」「制約」「誰のためか」「今の仮説」を書く
- もし情報が薄い場合でも、AIは仮説として語彙を提案し、追加で聞くべき情報も提示します

## 出力の受け取り方（おすすめ）
- まずは用語を **捨てずに** 収集
- 次に Top5〜10 に圧縮して調査を開始
- 学んだ語彙を課題文に追記 → もう一度 `/vocab` すると語彙がさらに良くなります
