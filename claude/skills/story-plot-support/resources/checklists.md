# チェックリスト（破綻防止・速度維持）

## 整合性（B系）
- [ ] タイムラインA（客観）に矛盾がない
- [ ] タイムラインB（提示）で"隠す/誤誘導/後出し"が明示されている
- [ ] 各重要情報に根拠シーンIDがある

## 伏線
- [ ] 全伏線に Plant と Payoff がある（孤児なし）
- [ ] Payoffの"証明方法"（台詞/映像/行動）が書かれている
- [ ] red herring の回収（誤誘導の解除）が弱くない

## テンポ（D系）
- [ ] 8シークエンスで同種（説明/会話/移動）が連続して停滞していない
- [ ] S4（中盤前）とS6（どん底前）が定義されている

## シーン強度（短期）
- [ ] 全シーンに Goal / Conflict / Turn / Button がある
- [ ] Value Shift がある（変化が起きる）
- [ ] "変わらないシーン"は統合 or 削除対象

## 群像運用（A系）
- [ ] 章×筋マトリクスで空白が連続しすぎてない
- [ ] 各章で最低1筋が前進している

## 書けない問題（E系）
- [ ] 打ち止めラインを越えて設計を続けていない
- [ ] ステップアウトラインがある（=書き始められる）
