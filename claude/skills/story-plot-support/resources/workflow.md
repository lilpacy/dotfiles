# 運用フロー（最短で"書ける状態"にする）

## A. 迷ったらこの順（推奨フロー）
1) 長期：ログライン → 終点1段落 → タイムラインA/B → 伏線台帳
2) 中期：8シークエンス → （必要なら）三幕/STCで節チェック →（必要なら）Truby/McKee点検
3) 短期：ステップアウトライン → シーンカード → 各シーンをG/C/T/B + Value Shiftで磨く

## B. 打ち止めライン（設計過多対策）
- 長期：終点1段落が書けたらOK、詳細はTBDで進む
- 中期：8シークエンスが揃ったら、短期へ降りる
- 短期：ステップアウトラインが全列挙できたら、執筆へ入る
