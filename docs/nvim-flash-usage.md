# flash.nvim 使い方

画面内の高速移動プラグイン。検索文字を入力するとラベルが表示され、一発でジャンプできる。

## キーマップ

| キー | モード | 説明 |
|------|--------|------|
| `s` | n, x, o | Flash ジャンプ |
| `S` | n, x, o | Flash Treesitter（構文単位でジャンプ） |

## 基本的な使い方

### sキーでジャンプ

1. `s` を押す
2. ジャンプ先の文字を1〜2文字入力
3. 表示されたラベル（a, b, c...）を押してジャンプ

例: 画面内の `function` にジャンプしたい場合
```
s → fu → [ラベルを押す]
```

### Sキーで構文単位ジャンプ

1. `S` を押す
2. Treesitterで認識された構文要素（関数、クラス、文など）にラベルが表示される
3. ラベルを押してジャンプ

## オペレーターと組み合わせ

Flash はオペレーターモード（`o`）でも動作するため、以下のような操作が可能：

| 操作 | 説明 |
|------|------|
| `ds[文字]` | Flash位置まで削除 |
| `ys[文字]` | Flash位置までyank |
| `cs[文字]` | Flash位置まで変更 |

例: カーソル位置から `return` まで削除
```
d → s → re → [ラベル]
```

## Visual モードでの選択

1. `v` でビジュアルモード開始
2. `s` を押して移動先を指定
3. カーソル位置から移動先までが選択される

## Tips

- 大文字小文字は区別される（`smart_case` がデフォルトで有効）
- 一意に特定できる場合はラベル入力不要で即ジャンプ
- `<Esc>` でキャンセル
