# outline.nvim 使い方

TSX/JSXのDOM構造を階層表示するためのプラグイン。aerialとは別で併用。

## 開閉

| キー | 操作 |
|------|------|
| `<leader>O` | Outlineウィンドウの開閉 |

## Outlineウィンドウ内の操作

### ジャンプ

| キー | 操作 |
|------|------|
| `<CR>` / クリック | シンボル位置にジャンプ |

### 個別の折りたたみ

| キー | 操作 |
|------|------|
| `h` | カーソル位置を折りたたむ |
| `l` | カーソル位置を展開 |
| `Tab` | カーソル位置をトグル |

### 一括折りたたみ

| キー | 操作 |
|------|------|
| `W` | 全て折りたたむ |
| `E` | 全て展開する |
| `R` | デフォルト状態にリセット |
| `Shift+Tab` | 全トグル（展開↔折りたたみ） |

## aerialとの使い分け

- `<leader>o` (aerial) - 関数/クラス/Markdownのシンボル表示
- `<leader>O` (outline) - TSX/JSXのDOM構造表示
