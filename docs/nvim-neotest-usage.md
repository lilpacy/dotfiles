# neotest 使い方

neotestはNeovim内でテストを実行・管理するためのフレームワーク。

## キーマップ

| キー | 説明 |
|------|------|
| `<leader>tt` | カーソル位置の最寄りのテストを実行 |
| `<leader>tf` | 現在のファイル全体のテストを実行 |
| `<leader>ts` | テストサマリーパネルの表示/非表示 |
| `<leader>to` | テスト出力を表示（結果詳細） |

## 基本的な使い方

### 単一テストの実行
1. テスト関数内にカーソルを置く
2. `<leader>tt` を押す
3. テストが実行され、結果がサインカラムに表示される

### ファイル全体のテスト実行
1. テストファイルを開く
2. `<leader>tf` を押す
3. ファイル内の全テストが実行される

### 結果の確認
- `<leader>ts` でサマリーパネルを開く（テスト一覧と状態）
- `<leader>to` で詳細な出力を確認（エラーメッセージなど）

## サポートしているテストフレームワーク

現在の設定では以下をサポート：
- **Jest** (JavaScript/TypeScript)

## Tips

- テストサマリーパネルでは `r` で再実行可能
- 失敗したテストは赤いサインで表示される
- 成功したテストは緑のサインで表示される
