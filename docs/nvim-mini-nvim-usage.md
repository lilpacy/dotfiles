# mini.nvim 操作方法

## mini.surround

テキストを括弧や引用符で囲む操作。

### 基本操作

| 操作 | コマンド | 例 |
|------|----------|-----|
| 追加 | `sa` + motion + char | `saiw"` → 単語を`"`で囲む |
| 削除 | `sd` + char | `sd"` → 周囲の`"`を削除 |
| 置換 | `sr` + old + new | `sr"'` → `"`を`'`に置換 |

### 使用例

```
hello  →  saiw"  →  "hello"
"hello"  →  sd"  →  hello
"hello"  →  sr"'  →  'hello'
```

**注意**: `saiw"` は `s` `a` `i` `w` `"` と順番に押す。一気に入力しても可。

### 対象指定 (motion)

- `iw` - inner word (単語)
- `aw` - a word (単語+空白)
- `i)` - 括弧内
- `a)` - 括弧含む
- `ip` - パラグラフ

## mini.pairs

括弧の自動補完。設定不要で動作。

- `(` → `()` (カーソルは中央)
- `{`, `[`, `"`, `'` も同様

## mini.ai

拡張テキストオブジェクト。

| オブジェクト | 説明 |
|--------------|------|
| `a)` / `i)` | 括弧（次の括弧も検索） |
| `aa` / `ia` | 関数引数 |
| `af` / `if` | 関数呼び出し |

### 使用例

```lua
func(arg1, arg2, arg3)
      ^カーソル
```

- `daa` → `arg1, ` を削除
- `cia` → `arg1` を変更
