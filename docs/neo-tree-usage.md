# neo-tree 使い方ガイド

## 基本操作

| キー | 機能 |
|------|------|
| `<C-n>` | neo-tree トグル |
| `q` | neo-tree を閉じる |
| `R` | リフレッシュ |
| `?` | ヘルプ表示 |

## ファイルを開く

| キー | 機能 |
|------|------|
| `<CR>` / `l` / クリック | ファイルを開く |
| `t` | 新しいタブで開く |
| `v` | 垂直分割で開く |
| `o` / `s` | 水平分割で開く |
| `P` | プレビュー切替 |

## ナビゲーション

| キー | 機能 |
|------|------|
| `h` | フォルダを閉じる |
| `-` | 親ディレクトリに移動 |
| `z` | 全フォルダを閉じる |
| `Z` | 全フォルダを開く |
| `/` | ファジーファインダー |
| `H` | 隠しファイル表示切替 |

## ファイル操作

| キー | 機能 |
|------|------|
| `a` / `%` | 新規ファイル作成 |
| `A` / `d` | 新規ディレクトリ作成 |
| `r` | リネーム |
| `D` | 削除 |
| `y` | コピー（クリップボード） |
| `x` | カット（クリップボード） |
| `p` | ペースト（クリップボード） |
| `c` | コピー（neo-tree内） |
| `m` | 移動（neo-tree内） |

## Git操作

| キー | 機能 |
|------|------|
| `ga` | ファイルをステージ (git add) |
| `gu` | ファイルをアンステージ |
| `gs` | Gitステータス表示 |
| `gc` | コミット |
| `gp` | プッシュ |
| `[g` | 前のGit変更ファイルへ |
| `]g` | 次のGit変更ファイルへ |

## 検索・フィルタ

| キー | 機能 |
|------|------|
| `/` | ファジーファインダー |
| `f` | フィルタ入力 |
| `<C-x>` | フィルタクリア |
| `<C-S-h>` | grug-farで検索・置換 |

## 右クリックメニュー

neo-tree上で右クリックすると以下のメニューが表示される:

- **コピー系**: ファイル名 / 相対パス / 絶対パス / ファイル内容
- **開く**: 通常 / 垂直分割 / 水平分割 / 新タブ
- **編集**: リネーム / 削除 / 新規作成
- **クリップボード**: カット / コピー / ペースト
- **検索**: grug-farで検索・置換
- **外部連携**: Finderで開く / Lazygit / GitHubをブラウザで開く

## 表示設定

### 非表示になっているもの（デフォルト）
- `.git` ディレクトリ
- `node_modules` ディレクトリ
- `.gitignore` に記載されたファイル

`H` キーで隠しファイルの表示を切り替えられる。

### Gitステータスの色
- 緑: 追加されたファイル
- 黄: 変更されたファイル
- 赤: 削除されたファイル

### アイコン
| アイコン | 意味 |
|---------|------|
| ✚ | 追加 |
|  | 変更 |
| ✖ | 削除 |
|  | リネーム |
|  | 未追跡 |
|  | ステージ済み |
|  | コンフリクト |
| ● | 未保存 |

## 幅の調整

- マウスドラッグで幅を変更可能
- 変更した幅は保持される
- `:NeoTreeResetWidth` でデフォルト(30)にリセット

## コマンド

| コマンド | 機能 |
|---------|------|
| `:Neotree toggle` | neo-tree トグル |
| `:Neotree reveal` | 現在のファイルを表示 |
| `:Neotree git_status` | Gitステータス表示 |
| `:Neotree buffers` | バッファ一覧表示 |
| `:NeoTreeResetWidth` | 幅をリセット |
