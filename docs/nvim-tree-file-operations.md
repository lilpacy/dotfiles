# nvim-treeでのファイル操作

nvim-treeでVSCodeのようにファイルを簡単に移動・コピーする方法。

## 基本操作

### ファイルの移動
1. 移動したいファイル/ディレクトリにカーソルを合わせる
2. `x` でカット
3. 移動先のディレクトリに移動
4. `p` でペースト

### ファイルのコピー
1. コピーしたいファイル/ディレクトリにカーソルを合わせる
2. `c` でコピー
3. コピー先のディレクトリに移動
4. `p` でペースト

## その他の便利な操作

### 複数ファイルの一括操作
1. `m` でファイルをマーク（複数選択可能）
2. `x` または `c` でまとめてカット/コピー
3. 移動先で `p` でペースト

### ファイル作成・削除・リネーム
- `%` - 新しいファイル作成
- `d` - 新しいディレクトリ作成（末尾に `/` をつける）
- `D` - 削除
- `R` - リネーム

## 現在のキーマッピング（カスタム設定）

`~/.config/nvim/lua/plugins/nvim-tree.lua:54-77` で定義:

- `<CR>` または `<LeftRelease>` - ファイルを開く
- `t` - タブで開く
- `v` - 垂直分割で開く
- `o` - 水平分割で開く
- `-` - 親ディレクトリに移動
- `<C-n>` - nvim-treeのトグル（グローバルキーマップ）

## 参考

デフォルトのキーマッピング一覧は `:help nvim-tree-mappings` で確認可能。
