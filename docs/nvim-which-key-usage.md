# which-key.nvim 使い方

キーマップのヘルプをポップアップ表示するプラグイン。キーを押して少し待つと、続けて押せるキーの一覧が表示される。

## 基本動作

1. `<leader>` などのプレフィックスキーを押す
2. 約300ms待つとポップアップが表示される
3. 次に押せるキーと説明が一覧で見える

## 主な表示タイミング

| プレフィックス | 表示される内容 |
|---------------|---------------|
| `<leader>` | リーダーキーに続くマッピング一覧 |
| `g` | `g` から始まる操作（`gd`, `gr`, `gi` など） |
| `z` | 折りたたみ、スペルチェックなど |
| `[` / `]` | 前/次へ移動系のマッピング |
| `"` | レジスタ一覧 |
| `'` / `` ` `` | マーク一覧 |

## ポップアップ内の操作

| キー | 説明 |
|------|------|
| `<Esc>` | ポップアップを閉じる |
| `<BS>` | 一つ前のキー入力を取り消し |
| `<Down>` / `<Up>` | スクロール |

## コマンド

```vim
:WhichKey           " 全マッピングを表示
:WhichKey <leader>  " <leader>から始まるマッピングを表示
:WhichKey g         " gから始まるマッピングを表示
```

## Tips

- 新しいキーマップを覚えるのに便利
- `desc` を設定したキーマップは説明付きで表示される
- 設定不要で自動的にマッピングを検出
