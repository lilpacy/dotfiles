# Web Doc Reading

- Prefer DuckDuckGo for search because Google often blocks bots.
- When reading documents, do not read raw HTML directly.
- Use `npx curl.md` to fetch pages as markdown:

```bash
npx curl.md https://example.com
```

- For current technical facts, prefer primary sources such as official documentation, release notes, standards, or source repositories.
